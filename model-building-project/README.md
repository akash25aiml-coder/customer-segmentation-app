#  Customer Segmentation App

An interactive Machine Learning web app built using Streamlit that segments customers based on their income and spending behavior using K-Means clustering.

##  Features

- 📂 Upload your own CSV dataset
- 📊 Automatic clustering using K-Means
- 🎯 Select number of clusters (interactive slider)
- 📈 Visualize customer segments
- 📍 Centroid visualization
- 📉 Elbow method to find optimal clusters
- 📊 Cluster distribution analysis
- 🔍 Explore individual clusters
- ⬇️ Download clustered dataset

---

##  Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

---

##  Dataset Requirements

The dataset must contain the following columns:

- `Annual Income (k$)`
- `Spending Score (1-100)`

---

##  How It Works

- Data is scaled using StandardScaler
- K-Means clustering is applied
- Users can select number of clusters
- Results are visualized and analyzed
